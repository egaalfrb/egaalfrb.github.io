Telkom Property Internship Project

Project ini dikerjakan selama masa magang di PT Graha Sarana Duta (Telkom Property) pada divisi IT & Digitization. Fokus utama dari project ini adalah mengintegrasikan website ESS absensi ke aplikasi mobile dan mengimplementasikan sistem deteksi Fake GPS untuk meningkatkan keandalan absensi.

Fitur & Kontribusi Utama

  • Integrasi Mobile: Mengintegrasikan website Employee Self Service (ESS) absensi ke dalam aplikasi mobile (WebView).
  • Fake GPS Detection: Merancang dan mengimplementasikan sistem deteksi aplikasi mock location/fake GPS dan location permission pada modul absensi.
  • Mekanisme Validasi: Mengatur mekanisme validasi lokasi saat melakukan login/absensi untuk menjaga keaslian data koordinat.
  • Keamanan Data: Mengelola keamanan dan pembatasan akses data absensi sesuai batasan untuk karyawan magang.

Uji Coba Demo Aplikasi (Tutorial Usil) 🤫

Penasaran mau coba mengelabui detektor kami? Ikuti langkah-langkah lucu berikut:

1

Buka Gerbang 🚪

Klik tombol demo di bawah & izinkan akses lokasi. Tenang, kami ga bakal kepoin kamu lagi nongkrong di mana!

2

Jalur Alim 😇

Buka dalam kondisi GPS normal (jujur). Sistem akan mendeteksi kamu aman sentosa. Lampu hijau menyala!

3

Mulai Nakal 😈

Aktifkan Fake GPS di HP kamu, atau tirukan mock location lewat Developer Tools browser (Sensors -> Location).

4

Terciduk Basah! 🚨

Seketika layar akan berubah merah menyala dan membongkar kepalsuan lokasimu. Kamu resmi tertangkap basah!

Web Portfolio Personal

Pembuatan website portfolio interaktif yang merangkum pendidikan, pengalaman, dan project. Website ini mengedepankan performa, responsivitas di berbagai perangkat, serta tampilan modern.

Fitur & Pengembangan

  • Modern UI/UX: Desain antarmuka dengan gaya bersih, elemen melengkung, hover effects interaktif, dan navigasi smooth-scrolling.
  • Fully Responsive: Tampilan yang menyesuaikan sempurna dari desktop hingga smartphone.
  • Custom Components: Penggunaan custom modal, lightbox gallery, scroll-spy untuk navigasi, dan efek pengetikan animasi (*typing effect*).
